Web App

Using docker images, I put the AWS pipeline on Jupyter notebook and deployed it using Heroku at classifydigit.herokuapp.com. This essentially creates a web app where the project is hosted and anyone can upload an image and test the system. The only caveat being the professor I was working with made us password protect access to the Jupyter notebook on the web app as we use his account to run the AWS computations. Meaning, if the web app was hosted publicly and someone spammed the site, he could be potentially charged a lot of money for the computation. Nonetheless, I do have a short video demo of the web app.
